# DevMedia Theme

![DevMedia](screenshots/Devmedia.jpeg)

Este tema foi criado por um estudante da DevMedia como iniciativa pessoal, inspirado nas cores e identidade visual da plataforma. **Não é um tema oficial da DevMedia.**
