# DevMedia Theme - Exemplos de Código

Esta pasta contém exemplos completos em diversas linguagens para testar o tema **DevMedia**.

## Estrutura

```
examples/
├── python/         # FastAPI REST API
├── javascript/     # Express.js API
├── typescript/     # Task Management API
├── java/           # Banking system
├── csharp/         # E-commerce order processing
├── go/             # URL shortener service
├── rust/           # CLI todo app
├── php/            # Blog engine
├── ruby/           # Weather data processor
├── cpp/            # Math library
├── html/           # Página HTML completa
├── css/            # Folha de estilos moderna
├── sql/            # E-commerce database queries
├── shell/          # Script de setup
├── swift/          # Task manager com async/await
├── kotlin/         # Note-taking app
├── docker/         # Dockerfile + docker-compose
├── data/           # JSON, YAML, Kubernetes
└── README.md
```

## Como usar

Abra esta pasta no VS Code com o tema DevMedia ativado para visualizar a syntax highlighting em todas as linguagens.
